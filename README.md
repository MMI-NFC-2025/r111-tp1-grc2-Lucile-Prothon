# R111-2025-TP1

## Objectif : réaliser la page HTML de cette landing page

[Lien vers la maquette Figma](https://www.figma.com/design/Qpq3vTQsR0CYrQl2LtwPRD/Housemood?node-id=0-1&t=qAtyOjA4yjMaQjt4-1)

Suivez les étapes :

- Exportez les images de la page d'accueil en veillant à ce qu'elles soient optimisées et de bonne qualité.
- Créez un répertoire `assets` et organisez à l'intérieur les différents types d'image comme vu en TD.
- Créez ensuite un fichier `index.html` à la racine du projet dans lequel vous écrirez le code HTML de la page d'accueil.
- Veillez à ce que le code de la page soit valide et bien structuré.
- Faites au moins 6 commits sur l'ensemble du TP en les nommant de manière pertinente.

S'il vous reste du temps, commencez les autres pages.
